Interface GTK - Documentation

Dans le cadre de notre projet d’algorithme en langage C, nous avons conçu un système capable de réagir dynamiquement à des interactions visuelles en s’inspirant de la lampe emblématique de l’univers Pixar. L'idée est donc que notre système réagisse à des interactions visuelles de manière dynamique, donnant alors l'impression que la machine est "vivante". Pour ce faire, le projet s’articule autour de deux figures principales :

  • une figure imposée, consistant à suivre une couleur définie en temps réel
  • une figure libre, dans notre cas un "Robot Police" capable de détecter et suivre un individu selon deux critères sélectionnés en amont

Ainsi, voici l'interface utilisateur qui permet de contrôler ces deux modes et leurs options :

  • Suivi de couleur (rouge, bleu, vert)

  • Recherche d’un suspect selon la couleur de son tee-shirt et port de masque

    L’interface développé avec GTK permet d'exécuter les deux programmes externes liés à ces modes selon les paramètres choisis.

Fichiers clés

  • interface.cpp : Point d’entrée principal de l’interface
  • couleur.txt / config.txt : Fichiers utilisés comme entrées pour les programmes externes, ils sauvegardent les paramètres.
  • camera_red_detect / robot_police_cam_mac : Programmes exécutés selon le mode
