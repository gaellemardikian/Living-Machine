◆ main()

int main ( )

Point d'entrée du programme.

Returns
0 en cas de succès, 1 si caméra ou port série est inaccessible.
  • Initialise la capture vidéo via la webcam par défaut (indice 0).
  • Ouvre la communication série avec l'Arduino.
  • Recherche les contours dans le masque.
  • Prends le contour le plus grand.
  • Ignore les petits objets.
  • Ajustement des angles pour recentrer l'objet détecté
  • Affiche un cadre central de référence
  • Quitter avec la touche ESC
    • Ferme le Port série
    • Libère la webcam.
    • Ferme les fenêtres OpenCV.
